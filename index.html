隐私政策

发布日期：2026年4月

OrbitShell 极度重视您的隐私。您的所有 SSH 密钥、密码、服务器地址均经过强加密，并**仅保存在您的本地磁盘**。

1. 数据本地化存储

我们不会上传、收集或出售您的任何敏感凭证。所有数据通过 macOS Keychain 进行安全管理。

2. AI 功能与交互

使用 AI 诊断时，仅发送您选定的文本至 API 服务商。我们建议您在发送前脱敏日志中的私密数据。
