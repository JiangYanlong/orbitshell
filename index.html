OrbitShell 隐私政策

发布日期：2026年4月

OrbitShell 极度重视您的隐私。作为一款专业的连接管理工具，我们坚守以下原则：

1. 数据本地化存储

您的所有 SSH 密钥、密码、服务器地址均经过强加密，并**仅保存在您的本地磁盘**。OrbitShell 不会上传、收集或出售您的任何敏感凭证。

2. AI 功能与交互

使用 AI 诊断时，仅发送您选定的文本至 API 服务商。我们建议您在发送前脱敏日志中的私密数据。

3. 安全性

我们使用 macOS 原生的 Keychain 保护敏感数据，确保即便设备遗失，数据也难以被第三方窃取。

如有疑问，请通过 support@orbitshell.cn 联系我们。
